# Unicloud - フリーランスエンジニア兼SEサービス事業 公式サイト

## 1. プロジェクト概要
フリーランスエンジニア「Unicloud」の公式ホームページ。屋号は「Unicloud」。「技術と人をつなぐ。」をコンセプトに、フリーランスエンジニア事業とエンジニア紹介事業の2軸で事業展開。クライアント企業への信頼感と技術力をアピールするランディングページ型の公式サイト。

## 2. ページ構造
- `/` - ホーム（1ページ構成、全セクションをスクロール表示）
  - ホーム（Hero）
  - 事業内容
  - 新着情報
  - 経歴
  - お問い合わせフォーム

## 3. コア機能
- [ ] Hero セクション（タグライン・CTAボタン）
- [ ] 事業内容セクション（2つの事業紹介）
- [ ] 新着情報セクション（お知らせ + noteリンク）
- [ ] 経歴セクション（職歴タイムライン + 資格一覧）
- [ ] お問い合わせフォーム（名前・メール・件名・内容 必須、電話番号・会社名 任意）
- [ ] スムーズスクロールナビゲーション

## 4. データモデル設計
現時点ではデータベース不要（静的コンテンツ + フォーム送信のみ）

## 5. バックエンド / サードパーティ連携計画
- フォーム送信: Readdyフォーム機能（お問い合わせ）
- Supabase: 不要
- Shopify: 不要
- Stripe: 不要

## 6. 開発フェーズ計画

### Phase 1: 全セクションの静的ページ構築
- 目標: 5セクションすべてを含む1ページの公式サイトを完成させる
- 成果物: 動作するランディングページ（全セクション・ナビゲーション・フォーム含む）
